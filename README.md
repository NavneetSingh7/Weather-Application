# 🌦️ Weather Application

This is a simple and responsive **Weather Web App** built using **HTML, CSS, and JavaScript**, which fetches real-time weather data using the **OpenWeatherMap API**.

---

## 🚀 Features

- 🔍 Search weather by city name
- 🌡️ Real-time temperature, humidity, and wind speed
- 🌤️ Dynamic weather icons based on condition (Rain, Clear, Clouds, Snow, etc.)
- 📱 Fully responsive design for mobile and desktop
- ❌ Error handling for invalid city names

---

## 🛠️ Tech Stack

- **HTML5**
- **CSS3**
- **JavaScript (Vanilla)**
- **OpenWeatherMap API**
